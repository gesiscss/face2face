face to face interaction toolkit

The face to face interaction toolbox provides an easy acccessible way to analyse social behavior based on face to face interaction data sets.

The toolbox provides:

  • Import of own and predefined face to face interaction data
  • Creating different kinds of networks
  • Statistical analysis of the networks and the raw interaction data
  • Visualisation of the results

The toolbox should enable everyone to analyse face to face interaction. To ensure that many tutorials and an online documentation are provided and linked below.

You can also take a look at the repository to get a deeper understanding of the underlying structure and the functions of this toolbox.

The PyPi Page contains further information about different versions of the toolbox and how you can install it. A standard installation guide is also provided in the online documentation
